# 微课管理平台 · 功能测试总计划

> **测试目标**：发现功能 Bug、表单/流程缺陷、权限越权、边界值遗漏
> **测试方法**：黑盒（浏览器端到端）+ 白盒（代码审计）双管齐下
> **环境**：本地开发环境（`localhost:5173` + `localhost:8080` + 本地 PG/Redis）
> **测试账号**：admin / admin123、teacher / teacher123、student / student123（README 给出）

---

## 一、模块清单（按优先级分 4 批）

### 🥇 第一批：核心流程（必测，1-2 小时）
1. **认证模块** — 登录 / 登出 / 错误密码 / 空字段 / 角色跳转
2. **数据看板** — 管理员 Dashboard 加载、统计数字是否正确
3. **课程管理** — 课程列表 / 搜索 / 详情 / 新增 / 编辑 / 删除 / 上下架
4. **章节视频** — 章节树 / 视频列表 / 视频上传 / 视频播放
5. **用户管理** — 用户列表 / 新增 / 编辑 / 启停用 / 角色分配

### 🥈 第二批：业务功能（2-3 小时）
6. **内容资源** — 题库 / 练习 / 试卷
7. **教务管理** — 部门 / 班级 / 排课
8. **微专业** — 微专业列表 / 学员管理 / 教师分配
9. **评价与评论** — 评价列表 / 评分 / 审核
10. **订单与购物车** — 购物车 / 下单 / 订单管理
11. **证书与徽章** — 证书模板 / 颁发 / 徽章规则

### 🥉 第三批：扩展功能（2-3 小时）
12. **互动插件** — 互动课程 / Slide
13. **讨论区** — 帖子 / 评论 / 管理
14. **通知系统** — 系统通知 / 偏好
15. **举报与审核** — 举报处理
16. **收藏 / 书签 / 笔记** — 学习侧
17. **AI 讲述稿** — DeepSeek 集成（如果配了 key）
18. **运营位 / Banner** — 平台 Banner
19. **操作日志** — 日志查询 / 导出

### 🏅 第四批：边界与权限（1-2 小时）
20. **表单校验** — 所有表单的边界值（空、超长、特殊字符）
21. **权限隔离** — 用 student 账号访问 admin 路径
22. **错误处理** — 网络断开 / 500 错误 / 401 token 过期
23. **分页/排序/筛选** — 数据量大时表现
24. **导出功能** — Excel / CSV 导出
25. **文件上传** — 大文件 / 错误格式

## 二、测试方法

| 阶段 | 工具 | 输出 |
|---|---|---|
| 黑盒（端到端） | Playwright（已配）+ 浏览器 | 截图、复现步骤 |
| 接口验证 | curl / 浏览器 Network | 状态码、响应体 |
| 白盒（代码） | 读 Controller/Service/Mapper | 逻辑漏洞、未覆盖校验 |
| 数据库验证 | psql 直查 | 数据一致性、事务回滚 |

## 三、Bug 报告规范

每发现一个 Bug 记录到 `01-bugs-found.md`，字段：

| 字段 | 说明 |
|---|---|
| 编号 | BUG-001 / P0~P2 |
| 级别 | P0 数据安全 / P1-C 客户可见 / P1-I 内部 / P2 建议 |
| 模块 | 例如"课程管理-新增" |
| 复现步骤 | 一步步写清楚 |
| 预期结果 | 正常应该怎样 |
| 实际结果 | 实际出了什么 |
| 截图/证据 | 路径 |
| 根因 | 代码层引用（白盒时填） |
| 修复建议 | 怎么改 |
