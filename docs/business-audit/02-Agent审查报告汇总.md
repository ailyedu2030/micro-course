## 交叉审批模式缺陷

审计发现一种系统性模式缺陷：**分片审查导致跨链路数据流断裂未被发现**。红队复核补充了 **14 个**跨链路问题（详见 `03-红队复核补充问题清单.md`），这些问题均是各 Agent 在各自链路内无法发现的。

| 模式 | 出现频次 | 示例 |
|------|:--------:|------|
| 数据流完整性断裂 | 3 | 线下签到→进度→统计三级连锁（R04） |
| 跨链异常处理断裂 | 2 | 微专业自动选课吞异常（R01） |
| 上下游断裂 | 2 | 审批通过状态阻断编辑（Admin R05） |
| 背景条件崩塌 | 2 | 分类删除后课程变成孤儿引用（Admin R02） |
| 责任边界盲区 | 2 | 成绩锁定每个模块都认为责任在对方（Admin R04） |
