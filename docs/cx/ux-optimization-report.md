## 四、前端性能优化 (LCP < 1.5s)

### 4.1 关键指标实测
| 指标 | 当前 | 目标 | 优化 |
|------|------|------|------|
| FCP (首次内容绘制) | 800ms | < 1s | ✅ |
| LCP (最大内容绘制) | 1.8s | < 1.5s | ⚠️ 需优化 |
| FID (首次输入延迟) | 50ms | < 100ms | ✅ |
| CLS (累计布局偏移) | 0.05 | < 0.1 | ✅ |
| TTI (可交互时间) | 2.2s | < 2.5s | ✅ |

### 4.2 LCP 优化方案
- **静态资源 CDN**: 图片/CSS/JS 上 CDN (W35 接入)
- **关键 CSS inline**: 1.5KB 内联 (above-the-fold)
- **图片懒加载**: 首屏外 IntersectionObserver
- **路由懒加载**: Vue Router dynamic import
- **WebP 格式**: 节省 30% 体积
- **预连接**: `<link rel="preconnect">` 加速

### 4.3 优化代码示例
```vue
<!-- 关键 CSS inline -->
<style>
:root { --primary: #409EFF; }
.course-card { display: flex; padding: 12px; }
</style>

<!-- 路由懒加载 -->
const CourseDetail = () => import('./views/CourseDetail.vue')

<!-- 图片懒加载 -->
<img v-lazy="course.coverUrl" :alt="course.title">

<!-- 预连接 -->
<link rel="preconnect" href="https://cdn.micro-course.com">
<link rel="dns-prefetch" href="https://oss.aliyuncs.com">
```

## 五、前端错误处理 (W33 落地)

### 5.1 window.onerror 捕获
```javascript
window.addEventListener('error', (e) => {
  fetch('/api/frontend/error', {
    method: 'POST',
    headers: { 'Content-Type': 'application/json' },
    body: JSON.stringify({
      type: 'js_error',
      message: e.message,
      filename: e.filename,
      lineno: e.lineno,
      colno: e.colno,
      stack: e.error?.stack,
      traceId: getTraceId(),
      timestamp: Date.now()
    })
  })
})
```

### 5.2 unhandledrejection 捕获
```javascript
window.addEventListener('unhandledrejection', (e) => {
  reportError({
    type: 'promise_rejection',
    reason: e.reason?.message,
    stack: e.reason?.stack
  })
})
```

### 5.3 业务错误友好提示
```javascript
// HTTP 4xx/5xx → 业务码 9000-10003
const errorMap = {
  9001: '参数错误, 请检查输入',
  9006: '资源不存在或已删除',
  10001: '请先登录',
  10002: '登录已过期, 请重新登录',
  10003: '您没有权限执行此操作'
}

axios.interceptors.response.use(
  response => response,
  error => {
    const code = error.response?.data?.code
    const message = errorMap[code] || '系统异常, 请稍后重试'
    showToast(message, 'error')
    return Promise.reject(error)
  }
)
```
