## 一、用户旅程 (8 阶段)

### 旅程 1: 教师上传 PPT 课件
| 阶段 | 操作 | 痛点 (前) | 优化 (后) | PR |
|------|------|----------|---------|-----|
| 1. 登录 | 输入账号密码 | 经常超时 | 记忆登录 30 天 | - |
| 2. 进入课件管理 | 左侧菜单点击 | 5 个职责混一组件 | **四面板工作台分流** | PR #38 |
| 3. 上传 PPT | 拖拽多张图片 | 一次性多页但无脚本 | 每页独立脚本编辑器 | PR #38 |
| 4. 写讲述稿 | 文本框输入 | 直接覆盖 | **版本永久保留 (V301 partial unique)** | PR #38 |
| 5. 生成音频 | 点击"AI 生成" | 等 5s 无反馈 | **状态可视化 (GENERATING/READY/FAILED)** | PR #38 |
| 6. 试听 | 点击播放 | 没声音无提示 | ElMessage.error + token 前 8 字符调试 | BUG #11 修复 |
| 7. 编辑页间跳转 | 配置 next/branch/skip | 缺失 | **PptFlowEditor (V306)** | PR #38 |
| 8. 保存发布 | 点击发布 | 经常 500 | traceId 报错可定位 | BUG #30 #31 修复 |
