### 建议避免的功能（专家一致反对）

| 功能 | 反对理由 |
|------|---------|
| 积分商城/虚拟货币 | 破坏学习内在动机（过度理由效应） |
| 3D 虚拟教室/元宇宙 | 技术成本高、学生需求低、带宽要求苛刻 |
| 复杂徽章系统（>20 种） | 超 70% 用户从不查看，维护成本高 |
| 强制社交（必须加群才能学） | 学生抵触，高校场景不合适 |
