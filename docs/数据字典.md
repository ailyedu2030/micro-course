## 1. 核心用户表

### 1.1 users — 用户表

> 学生、教师、管理员、教务处统一存储。角色通过 role 字段区分。

| 字段名 | DB 列 | Java 类型 | TS 类型 | 长度 | 约束 | 说明 |
|--------|-------|-----------|---------|------|------|------|
| id | id | Long | number | - | PK, 自增 | 主键 |
| username | username | String | string | 50 | UNIQUE, NOT NULL | 登录用户名 |
| password | password | String | string | 255 | NOT NULL | bcrypt 加密 |
| realName | real_name | String | string | 50 | NOT NULL | 真实姓名[存储加密，API 脱敏返回] |
| email | email | String | string | 100 | UNIQUE (WHERE NOT NULL AND deleted_at IS NULL) | 邮箱[存储加密，API 脱敏返回] |
| phone | phone | String | string | 30 | - | 手机号[存储加密，API 脱敏返回] |
| gender | gender | String | string | 10 | - | 性别：MALE / FEMALE / SECRET |
| avatar | avatar | String | string | 500 | - | 头像 URL |
| role | role | String | string | 20 | NOT NULL | STUDENT / TEACHER / ADMIN / ACADEMIC |
| departmentId | department_id | Long | number | - | FK→departments | 所属院系 ID |
| majorId | major_id | Long | number | - | FK→majors | 所属专业 ID |
| classId | class_id | Long | number | - | FK→classes | 所属班级 ID |
| grade | grade | String | string | 10 | - | 年级（如"2023"），冗余字段便于查询 |
| enrollmentYear | enrollment_year | String | string | 10 | - | 入学年份（如"2023"） |
| graduationYear | graduation_year | String | string | 10 | - | 预计毕业年份 |
| politicalStatus | political_status | String | string | 20 | - | 政治面貌（依据《个人信息保护法》第28条需单独授权同意） |
| studentNo | student_no | String | string | 30 | UNIQUE | 学号（学生） |
| teacherNo | teacher_no | String | string | 30 | UNIQUE | 工号（教师） |
| teacherStatus | teacher_status | Integer | number | 1 | default 0 | 教师入驻审核状态：0=待审核, 1=通过, 2=驳回（V24 新增） |
| status | status | Integer | number | 1 | NOT NULL, default 1 | 用户状态：0=INACTIVE, 1=ACTIVE, 2=DISABLED, 3=DELETED |
| casBound | cas_bound | Boolean | boolean | - | default false | 是否绑定 CAS |
| lastLoginAt | last_login_at | LocalDateTime | string | - | - | 最后登录时间 |
| deletedAt | deleted_at | LocalDateTime | string | - | - | 软删除时间戳，非空表示已删除 |
| createdAt | created_at | LocalDateTime | string | - | NOT NULL | 创建时间 |
| updatedAt | updated_at | LocalDateTime | string | - | NOT NULL | 更新时间 |
| version | version | Integer | number | - | NOT NULL, default 0 | 乐观锁版本号（Round 6 新增，V63 migration） |

**索引**：idx_users_username (UNIQUE), idx_users_student_no (UNIQUE), uk_users_email (PARTIAL UNIQUE WHERE email IS NOT NULL AND deleted_at IS NULL), idx_users_role, idx_users_department, idx_users_major, idx_users_class, idx_users_deleted

**变更历史**：
- V63: 新增 version 字段
- V75: 新增 uk_users_email 部分唯一索引（兼容存量 NULL 邮箱）

## 附录：数据脱敏与安全策略

> 依据《个人信息保护法》《数据安全法》及行业规范制定。

### 1. 敏感字段脱敏规则

| 字段 | 存储要求 | API 返回脱敏 |
|------|----------|--------------|
| users.realName | AES-256 加密 | 姓保留，名掩码：张** |
| users.phone | AES-256 加密 | 前3后4掩码：138****1234 |
| users.email | AES-256 加密 | @前保留首字：j***@example.com |
| users.politicalStatus | 明文存储 | 仅授权用户可查看 |
| enrollments.finalScore | 明文存储 | 学生端仅返回等级制 |
| operation_logs.ip | 明文存储 | 段地址脱敏，保留前8位 |

### 2. 视频安全

| 策略 | 说明 |
|------|------|
| 播放签名(playSign) | HMAC-SHA256 动态签名，防盗链 |
| 签名有效期(signExpiredAt) | 默认 2 小时 |
| 水印(watermarkEnabled) | 叠加用户 ID + 时间戳 |
| 并发控制(maxPlayRate) | 限制单视频并发播放数 |

### 3. 附件访问控制

| 级别 | 说明 |
|------|------|

| PRESIGNED | 预签名 URL，有有效期 |

- 不支持物理删除，仅归档
