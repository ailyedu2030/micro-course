### 🟡 P1-C — 客户可感知

| ID | 缺陷 | 严重度 | 文件/端点 | 说明 |

| #1 | **视频上传缺 file 参数返回 500** | P1-C | `VideoController.upload()` line 109-115 | 当传了 `courseId` 但没传 `file` 时，Spring MVC 返回 500 Internal Server Error。应返回 400 参数校验错误。 |

### 🔵 P1-I — 内部仅见

| ID | 缺陷 | 严重度 | 文件/端点 | 说明 |
|----|------|--------|----------|------|
| #2 | **PUT /api/admin/settings 格式文档未明确** | P1-I | `AdminSettingsController.updateBatch()` | 需要 `[{...}]` 数组格式，单对象 `{...}` 返回 9005。API 契约文档中应标注请求体为数组。 |
| #3 | **POST /api/videos/{id}/play HLS 不可用时返回 500** | P1-I | `VideoController.play()` | 转码未完成返回 12004 但 HTTP 状态码为 500。建议改为 503 Service Unavailable。 |
| #4 | **/admin/settings/cas 密钥硬编码模式** | P1-I | AdminSettingsController | CAS 密码通过 `FieldEncryptor` 加解密，但 `superAdmins` 在内存中以明文拼接。 |

### 🟢 P2 — 代码整洁/建议

| ID | 缺陷 | 严重度 | 说明 |
|----|------|--------|------|
| #5 | **API 响应 HTTP 状态码不统一** | P2 | 部分错误使用 500，建议统一错误码与 HTTP 状态码映射（如 12004 → 503） |
| #6 | **操作日志总数数据较大(2878)** | P2 | 无自动清理机制，建议按时间/数据量做滚动清理或归档 |
