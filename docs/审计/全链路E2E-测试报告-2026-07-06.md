## 发现的阻断问题

### 问题 1：对话框中的 Element Plus Select 无法通过 Playwright 直接点击
- **严重程度**：P1-I（内部仅见）
- **描述**：在"新增课程"对话框中，分类等下拉选择框被 overlay/对话框遮罩层拦截，Playwright 的常规 click() 操作无法触达。报错信息为 ...intercepts pointer events。
- **影响**：自动化测试难以通过标准 UI 操作完成对话框填写
- **临时方案**：通过 page.evaluate() 直接触发 JS 事件操作 select 组件，或使用 API 方式绕过

### 问题 2：课程提交审核需要封面图
- **严重程度**：P1-C（客户可感知）
- **描述**：POST /api/courses/{id}/submit 如果课程没有封面图会返回"请先上传课程封面再提交审核"
- **影响**：教师在未上传封面时无法提交审核
- **当前行为**：错误提示合理，但建议给默认封面或提供更友好的引导

### 问题 3：学生课程详情页渲染异常
- **严重程度**：P1-C
- **描述**：访问 /student/courses/71 时页面显示"页面出了点问题"，而不是正常的课程详情页
- **影响**：学生无法通过课程广场中的课程卡片直接进入详情页
- **临时方案**：通过"我的课程"页面的"继续学习"按钮进入学习页面是正常的

### 问题 4：学生端快速登录标签不可点击
- **严重程度**：P2
- **描述**：登录页的"管理员、教务处、教师、学生"标签是纯展示的 el-tag，没有绑定点事件
- **影响**：用户期望点击标签能快速填充账号信息

### 问题 5：视频上传 MIME 类型检查严格
- **严重程度**：P2
- **描述**：POST /api/videos/upload 检查 MIME 类型必须为 video/*，默认curl上传会识别为 application/octet-stream
- **影响**：开发者用curl测试时需要显式指定 -F "file=@file.mp4;type=video/mp4"

### 问题 6：章节无练习模块
- **严重程度**：P1-I
- **描述**：创建的章节没有关联练习/习题数据，学习页面显示"练习 0/0 完成"
- **影响**：完整"做练习→拿证书"流程缺少数据支撑
