### 根本原因（5 Why 分析）

1. **Why 容器反复重启？** → 因为每次启动失败都用 `docker rm` + `docker run` 重试，而不是先在本地方案验证
2. **Why 不先验证？** → 因为 AI 把生产服务器当成 debug 环境，跳过 local-dev-deploy.sh 步骤
3. **Why 当成 debug 环境？** → 因为 `production-safety` skill 没有强制约束（仅是建议），且没有 5 分钟内禁止重复操作的具体规则
4. **Why 没有强制约束？** → 因为 AGENTS.md 没有 P0 级别的铁律章节
5. **Why AGENTS.md 没有？** → 历史遗漏，生产安全规则散落在不同 skill 中
