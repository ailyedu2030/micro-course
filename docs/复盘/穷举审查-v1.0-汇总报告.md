## 2. P0 阻塞项（必须修）

### P0-1 · 缺 Redis 配置 + 依赖（E5）

**问题**：
- `micro-course-api/pom.xml` 缺 `spring-boot-starter-data-redis` 依赖
- `micro-course-api/src/main/resources/application.yml` 缺 `spring.data.redis.*` 配置

**影响**：
- Phase 3 登录锁定（错误码 1006）无法实现
- Phase 3 JWT Token 黑名单无法实现
- 业务逻辑 §2.1 / §2.4 要求 Redis 计数 + 30 分钟锁

**修复方案**（总工程师建议）：
1. `pom.xml` 加 `spring-boot-starter-data-redis:3.2.5`
2. `application.yml` 加 `spring.data.redis.host/port/timeout/password`
3. 写 `RedisConfig.java`（Lettuce 连接池 + Jackson 序列化）
4. 启动后端验证 Redis ping 联通

**决策点**：是现在修，还是 Phase 3 开工前再修？
