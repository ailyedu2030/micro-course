## 一、为什么现在启动

微课平台面对 **K12 / 职教 / 语言 / 老年大学** 四类客群. 任何一次"为赶工而牺牲 UX"的决策 = 把用户赶向竞争对手. 一票否决不立, 用户就流失.

**项目历史已经验证**:
- W34 报告识别 8 项 P0 UI 缺陷 → W36+W37 全部补齐
- 2026-07-19 P0 jar-deploy-bypass → 设立 P0 生产安全铁律
- UX 治理体系 17 文件落地 → 但治理停留文档, 缺一个 "统率号召"

**本次启动**: 把 UX 治理从"文件齐备"升级到"运作机制生效".
