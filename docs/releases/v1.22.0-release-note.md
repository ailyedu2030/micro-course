## 🎯 业务影响（按客户场景）

### 场景 1: 学生购买套餐后想学选修课

| 项目 | 内容 |
|------|------|
| 修复前 | 套餐购买只发必修课访问权，选修课打不开（违反"买了就能学"承诺） |
| 修复后 | 套餐购买一次性发放所有课程访问权 |
| 客户路径 | 学生中心 → 我的课程 → 选修课入口 |

### 场景 2: 学生申请套餐退款

| 项目 | 内容 |
|------|------|
| 修复前 | 可能产生"钱退了但课还能学"的错账（事务边界不全） |
| 修复后 | 先回收全部访问权，成功后才落退款状态和流水；任一失败整笔回滚 |
| 客户路径 | 我的订单 → 申请退款 |

### 场景 3: 学生查看自己的订单

| 项目 | 内容 |
|------|------|
| 修复前 | 套餐订单显示成某门课的标题，跳转到单课详情 |
| 修复后 | 套餐订单显示套餐标题，列名改为"商品"，跳转套餐详情 |
| 客户路径 | 我的订单 → 套餐订单 |

### 场景 4: 教师查看课程学员

| 项目 | 内容 |
|------|------|
| 修复前 | 学员列表、统计、导出包含已取消/候补/退课学员，数据膨胀 |
| 修复后 | 统一排除 CANCELLED/WAITLIST/DROPPED/REJECTED 状态 |
| 客户路径 | 教师端 → 课程管理 → 学员列表 |

### 场景 5: 学生查看我的课程/学习中心

| 项目 | 内容 |
|------|------|
| 修复前 | 已退课/被驳回课程混入"进行中" |
| 修复后 | 6 个页面统一过滤，只展示有效在学课程 |
| 客户路径 | 我的课程 / 学习中心 / 训练中心 / 周报 |

### 场景 6: 匿名访问微专业详情

| 项目 | 内容 |
|------|------|
| 修复前 | 匿名访问白屏崩溃（数据源错位） |
| 修复后 | 正常显示微专业信息，按钮显示"登录后申请" |
| 客户路径 | 微专业广场 → 详情页 |

## 🔧 技术变更

### 后端 (11 处)

1. `OrderServiceImpl.enrollBundleCourses()` - 套餐购买发全部课程
2. `OrderServiceImpl.refund()` - 退款原子性（先回收权限）
3. `OrderServiceImpl.unenrollBundleCourses()` - 失败即时抛出
4. `OrderServiceImpl.toVO()` - 套餐订单返回套餐标题
5. `EnrollmentServiceImpl` - PAYMENT 渠道校验恢复 PENDING/PAID（回归修复）
6. `EnrollmentQueryServiceImpl` - 3 个查询方法加状态过滤
7. `CourseServiceImpl.computeStats()` - 统计口径对齐
8. `TeacherServiceImpl.getStats()` - 教师仪表盘统计修复
9. `MicroSpecialtyEnrollmentServiceImpl` - 角色收紧
10. `MicroSpecialtyProgressServiceImpl` - 结业条件修复
11. `MicroSpecialtyQueryServiceImpl` - 详情字段修正

### 前端 (14 处)

1. `utils/enrollmentFilters.js` - 新增统一过滤助手（**新建文件**）
2. `views/student/MyCourses.vue` - 状态过滤
3. `views/student/TrainingCenter.vue` - 状态过滤
4. `views/student/LearningCenter.vue` - 状态过滤
5. `views/student/WeeklyReport.vue` - 状态过滤
6. `views/student/BundleDetail.vue` - 已购判定修正
7. `components/profile/WrongQuestionsCard.vue` - 状态过滤
8. `views/student/MyOrders.vue` - 列名改"商品"，套餐跳套餐
9. `views/student/MicroSpecialtyDetail.vue` - 匿名不崩溃
10. `views/student/MyMicroSpecialties.vue` - COMPLETED 状态模板
11. `components/StudentLayout.vue` - 汉堡菜单断点
12. `views/public/MicroSpecialtySquare.vue` - 收藏图标映射
13. `router/index.js` - H5 标题回退
14. `utils/request.js` - 错误处理加固

### 测试 (5 处新增)

1. `OrderServiceBundlePriceTest` - 套餐全课程/展示/退款 3 个测试
2. `OrderPaymentFlowE2ETest` - 支付回归 4 个测试锁定
3. `MicroSpecialtyEnrollmentServiceTest` - 角色/容量 2 个测试
4. `MicroSpecialtyProgressAggregatorTest` - 聚合逻辑 1 个测试
5. `MicroSpecialtyQueryServiceTest` - 查询字段 1 个测试
