# 成绩回写接口规范 v1.0

> **状态**: P2 - 需对接教务系统，当前不可用
>
> **最后更新**: 2026-06-12

---

## 1. 概述

成绩回写功能用于将微课平台中的课程成绩（视频观看进度、练习得分、讨论参与度等）回写到外部教务系统。当前状态为**占位规范**，待与教务系统对接后方可实施。

---

## 2. 接口规范

### 2.1 单科成绩回写

#### 请求
```
POST /api/grade-writeback/courses/{courseCode}/students/{studentNo}
Content-Type: application/json
Authorization: Bearer {token}
```

```json
{
  "semester": "2026-1",
  "components": [
    {
      "type": "VIDEO_PROGRESS",
      "weight": 0.3,
      "score": 95.0,
      "detail": {
        "watchedDuration": 180,
        "totalDuration": 200,
        "completionRate": 0.90
      }
    },
    {
      "type": "EXERCISE_SCORE",
      "weight": 0.5,
      "score": 88.0,
      "detail": {
        "avgScore": 88.0,
        "totalExercises": 10,
        "passedExercises": 9
      }
    },
    {
      "type": "DISCUSSION_PARTICIPATION",
      "weight": 0.2,
      "score": 100.0,
      "detail": {
        "postCount": 5,
        "replyCount": 12
      }
    }
  ],
  "finalScore": 92.4,
  "finalGrade": "A",
  "completedAt": "2026-06-12T10:00:00Z"
}
```

#### 响应
```json
{
  "code": 200,
  "message": "ok",
  "data": {
    "studentNo": "2021001234",
    "courseCode": "CS101",
    "semester": "2026-1",
    "finalScore": 92.4,
    "finalGrade": "A",
    "writebackAt": "2026-06-12T10:00:00Z",
    "status": "SUCCESS"
  }
}
```

### 2.2 批量成绩回写

#### 请求
```
POST /api/grade-writeback/batch
Content-Type: application/json
Authorization: Bearer {token}
```

```json
{
  "semester": "2026-1",
  "courseCode": "CS101",
  "grades": [
    {
      "studentNo": "2021001234",
      "finalScore": 92.4,
      "finalGrade": "A"
    },
    {
      "studentNo": "2021001235",
      "finalScore": 85.0,
      "finalGrade": "B"
    }
  ]
}
```

#### 响应
```json
{
  "code": 200,
  "message": "ok",
  "data": {
    "totalCount": 100,
    "successCount": 98,
    "failCount": 2,
    "failures": [
      {
        "studentNo": "2021001240",
        "reason": "学生未选修该课程"
      }
    ]
  }
}
```

### 2.3 回写状态查询

#### 请求
```
GET /api/grade-writeback/status?semester=2026-1&courseCode=CS101
Authorization: Bearer {token}
```

#### 响应
```json
{
  "code": 200,
  "message": "ok",
  "data": {
    "semester": "2026-1",
    "courseCode": "CS101",
    "totalStudents": 100,
    "writtenBackCount": 98,
    "pendingCount": 2,
    "lastWritebackAt": "2026-06-12T10:00:00Z",
    "status": "PARTIAL" | "COMPLETED" | "PENDING"
  }
}
```

## 3. 成绩组成部分

| 组成部分 | 类型 | 权重 | 说明 |
|---------|------|------|------|
| VIDEO_PROGRESS | 视频进度 | 30% | 视频观看完成度 |
| EXERCISE_SCORE | 练习得分 | 50% | 练习平均得分 |
| DISCUSSION_PARTICIPATION | 讨论参与 | 20% | 发帖/回复数量 |

---

## 4. 等级划分

| 等级 | 分数范围 |
|------|----------|
| A | 90-100 |
| B | 80-89 |
| C | 70-79 |
| D | 60-69 |
| F | <60 |

---

## 5. 错误码

| 错误码 | 说明 |
|--------|------|
| 14001 | 教务系统不可用 |
| 14002 | 学生不存在 |
| 14003 | 课程不存在 |
| 14004 | 成绩回写失败 |
| 14005 | 重复回写（已存在成绩） |

---

## 6. 注意事项

- 所有回写接口需要管理员权限
- 回写前需校验学生是否选修该课程
- 最终成绩由各部分加权计算得出
- 回写成功后需记录回写时间和状态
- 支持部分成功场景，失败数据需记录原因
