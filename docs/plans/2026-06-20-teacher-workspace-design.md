# 教师课程工作台 · 设计方案 v1.0

> 定位：统一工作台，消除"散落页面"问题
> 设计参考：Notion/Figma/Teachable/Canvas/Gutenberg
> 原则：左侧大纲 = 结构，中间 = 内嵌编辑，右侧 = 上下文属性，顶部 = 发布

---

## 一、设计总览

### 1.1 核心问题

当前教师端课程管理功能分散在多个页面：

| 当前入口 | 页面 | 问题 |
|---------|------|------|
| 课程列表 | `/teacher/courses` | 表格混排，无类型区分 |
| 创建课程 | CourseList 弹窗 | 缺 courseType、price 字段 |
| 编辑课程 | `/courses/:id/edit` | 教师能看到审核按钮 |
| 幻灯片管理 | `/teacher/courses/:id/slides` | 无UI入口，需手输URL |
| 查看课程 | `/courses/:id` | 无课件管理入口 |

教师需要反复跳转，每次跳转丢失上下文，操作反人类。

### 1.2 统一工作台方案

```
┌──────────────────────────────────────────────────────────┐
│  ← 返回课程列表  │  课程名称   │  💾 自动保存  │  🔽 发布  │
├─────────────┬───────────────────────┬────────────────────┤
│   大纲树      │    内容编辑器          │    属性面板        │
│   (课时列表)  │   (按课程类型切换)     │  (上下文感知)      │
│             │                       │                   │
│  可拖拽排序   │  视频课→视频上传器     │  选中课时→课时属性  │
│  状态标识     │  互动课→幻灯片编辑器   │  选中章节→章节属性  │
│  进度计数     │  练习课→题库选择器     │  空白→课程元数据    │
│             │                       │                   │
│  章节·课时树  │  内嵌编辑·零跳转       │  底部进度概览       │
├─────────────┴───────────────────────┴────────────────────┤
│  📖 内容  │  👥 学员  │  📊 统计  │  ⚙️ 设置           │
└──────────────────────────────────────────────────────────┘
```

**关键设计原则**：

1. **大纲侧栏 = 课程结构**：不是导航链接，而是可拖拽的章节-课时树。选中某条目→中间编辑器切换。这节课的排序即课程本身的排序。
2. **中间画布 = 内嵌编辑器**：不跳转页面。编辑器按课程类型和课时类型自动切换。保存自动触发。
3. **右侧属性 = 上下文感知**：选中课时显示课时属性（标题/时长/文件/可见性），选中章节显示章节属性（名称/排序），空白处显示课程元数据。
4. **底部标签 = 视图切换**：内容/学员/统计/设置四个视图，大纲侧栏在所有视图中保持可见，不丢失上下文。
5. **顶部发布按钮 = 始终可见**：课程的草稿/已发布/待审核状态和操作始终在顶栏展示。

### 1.3 适用范围

本方案覆盖教师端课程管理的全部流程：

```
课程列表 → 创建工作台 → 编辑内容 → 审核 → 发布 → 查看统计
     ↑                                        |
     └────────────────────────────────────────┘
     (从任何课程直接进入工作台)
```

### 2.3 创建流程

点击「+ 新建课程」→ 弹出创建对话框：

```
┌─ 创建课程 ────────────────────────────────────────┐
│  课程标题:  [______________________________]    │
│  课程类型:  ○ 视频课程  ● 互动课程                │
│  课程分类:  [编程基础          ▼]                │
│  课程难度:  [初级    ▼]                          │
│  价格(元):  [0.00]  (0 = 免费)                  │
│                                                   │
│  [取消]                    [创建并进入工作台]      │
└───────────────────────────────────────────────────┘
```

**关键行为**：
- 创建后**不返回列表**，直接进入该课程的工作台
- 课程类型在创建时选定，后续可修改但会提示影响
- 创建时只填核心字段，其他元数据在工作台右侧属性面板中填写

```
│    ├📄 课时1 │  📊 INTERACTIVE→     │  章节属性           │

│  📖 内容  │  👥 学员  │  📊 统计  │  ⚙️ 设置            │
└──────────────────────────────────────────────────────────┘
```

### 3.2 三栏交互规则

| 交互 | 行为 |
|------|------|
| 左侧点击章节 | 右侧显示章节属性（名称、描述、排序） |
| 左侧点击课时 | 中间切换为该课时编辑器，右侧显示课时属性 |
| 左侧拖拽章节/课时 | 更新排序，自动保存 |
| 右侧属性编辑 | 编辑后自动保存（debounce 1s） |
| 中间编辑器操作 | 文件上传 / PPT解析 / 题库选择，所有操作在中间完成 |
| 顶部「发布」按钮 | 根据状态显示（草稿→提交审核 / 待审核→不可操作 / 已发布→下架） |

### 3.3 关键组件

| 组件 | 文件 | 说明 |
|------|------|------|
| `TeacherWorkspace.vue` | `views/teacher/Workspace.vue` | 三栏布局容器，管理选中状态 |
| `WorkspaceHeader.vue` | `components/teacher/workspace/Header.vue` | 顶栏（返回+名称+状态+发布） |
| `OutlineSidebar.vue` | `components/teacher/workspace/Sidebar.vue` | 左侧大纲树（章节+课时+拖拽+状态） |
| `ContentCanvas.vue` | `components/teacher/workspace/ContentCanvas.vue` | 中间编辑器容器（按类型分发） |
| `PropertiesPanel.vue` | `components/teacher/workspace/Properties.vue` | 右侧属性面板（上下文感知） |
| `WorkspaceTabs.vue` | `components/teacher/workspace/Tabs.vue` | 底部Tab切换 |

### 3.4 选中状态管理

```javascript
// 核心状态
const selectedItem = ref(null)  // { type: 'chapter'|'lesson', id: number }
const courseData = ref({})      // 完整课程数据
const courseStructure = ref([]) // 章节-课时树 [{ chapter, lessons:[] }]
const activeTab = ref('content') // content | students | stats | settings

// 选中改变 → 中间/右侧自动更新
watch(selectedItem, (item) => {
  if (item?.type === 'lesson') loadLessonEditor(item.id)
  else if (item?.type === 'chapter') loadChapterEditor(item.id)
  else loadCourseOverview()
})
```

### 3.5 自动保存

所有编辑操作（标题修改、拖拽排序、上传文件）触发自动保存：

- 编辑文字：`debounce 1000ms` 后调用 API
- 拖拽排序：拖拽结束后立即调用 `PUT /api/chapters/sort`
- 上传文件：上传完成后立即更新
- 状态切换：点击后立即调用

### 4.2 行内交互

| 元素 | 交互 |
|------|------|
| 章节标题 | 点击→选中章节，右侧显示章节属性 |
| 课时标题 | 点击→选中课时，中间切换编辑器 |
| ▶/▼ | 展开/折叠章节 |
| 📄 图标 | 视频课固定显示 📺，互动课显示 📊 |
| ✓/◌ | 已完成/未完成（有内容=已完成） |
| 拖拽把手 | 章节/课时可拖拽排序（从 ↔ 图标拖拽） |
| 右键菜单 | 右键→弹出操作菜单（复制课时/删除/移动到） |

### 4.3 课时类型图标

| 类型 | 图标 | 说明 |
|------|------|------|
| VIDEO | 📺 | 视频课时 |
| INTERACTIVE | 📊 | 互动课时（PPT+AI+TTS） |
| EXERCISE | 📝 | 练习课时 |
| TEXT | 📄 | 纯文字课时（可选） |

### 4.4 状态标记

| 状态 | 标记 | 含义 |
|------|------|------|
| 已完成 | ✓ 绿色 | 内容完整、有视频/有讲述稿 |
| 待完善 | ◌ 灰色 | 新建、内容为空 |
| 待配音 | 🎤 橙色 | 互动课时：PPT已上传、讲述稿已确认、音频未生成 |
| 音频就绪 | 🔊 绿色 | 互动课时：音频已生成 |

### 4.5 交互行为

```
拖拽：
  章节拖拽 → 调整章节顺序
  课时在章节内拖拽 → 调整课时顺序
  课时跨章节拖拽 → 移动到另一个章节
  
选中：
  点击章节 → 选中章节（右侧显示章节属性）
  点击课时 → 选中课时（中间+右侧同时切换）
  
行内操作：
  章节行末尾 [...] → 展开操作菜单
  课时行末尾 [...] → 展开操作菜单
```

### 5.1 课程类型分发

```javascript
// ContentCanvas.vue
// 根据 selectedItem.type 和 courseType 分发到不同编辑器
function getEditorComponent() {
  if (!selectedItem.value) return CourseOverview
  if (selectedItem.value.type === 'chapter') return ChapterEditor
  if (selectedItem.value.type === 'lesson') {
    switch (lessonType.value) {
      case 'VIDEO': return VideoLessonEditor
      case 'INTERACTIVE': return InteractiveLessonEditor
      case 'EXERCISE': return ExerciseLessonEditor
    }
  }
}
```

## 十一、数据流

### 11.1 加载流程

```
进入工作台
  → GET /api/courses/{id} (课程数据)
  → GET /api/chapters?courseId={id} (章节列表)
  → GET /api/videos?courseId={id} (视频列表，VIDEO课时用)
  → GET /api/courses/{id}/slides (幻灯片数据，INTERACTIVE课时用)
  → 构建 courseStructure 树
  → 渲染大纲侧栏
  → 默认选中第一个章节
```

### 11.2 保存流程

```
用户编辑 → debounce 1000ms → 自动保存
  - 文字属性：PUT /api/courses/{id} 或 PUT /api/chapters/{id}
  - 拖拽排序：PUT /api/chapters/sort
  - 上传文件：POST /api/videos/upload 或 POST /api/courses/{id}/slides/upload
  - 状态变更：POST /api/courses/{id}/submit 等
```

### 11.3 选中→编辑器映射

```
selectedItem = null           → CourseOverview（课程概览）
selectedItem.type = 'chapter' → ChapterEditor（章节编辑）
selectedItem.type = 'lesson' + courseType=VIDEO
                              → VideoLessonEditor（视频课时）
selectedItem.type = 'lesson' + courseType=INTERACTIVE
                              → InteractiveLessonEditor（互动课时）
selectedItem.type = 'lesson' + lessonType=EXERCISE
                              → ExerciseLessonEditor（练习课时）
```

## 十二、错误处理

### 12.1 加载错误

| 场景 | 行为 |
|------|------|
| 课程加载失败 | 显示错误页 + 「重试」按钮 |
| 章节加载失败 | 左侧大纲显示「加载失败」+ 重试 |
| 课时编辑加载失败 | 中间画布显示「加载失败」+ 重试 |
| 幻灯片加载失败 | INTERACTIVE编辑器显示「加载失败」+ 重试 |

### 12.2 保存错误

| 场景 | 行为 |
|------|------|
| 自动保存失败 | 图标变红 ⚠️ + 右下角toast「保存失败，请重试」 |
| 上传失败 | 上传区域显示错误信息 |
| 拖拽排序失败 | 自动回滚到原位置 + toast「排序失败」 |

### 12.3 权限错误

| 场景 | 行为 |
|------|------|
| 非课程拥有者访问工作台 | 显示403页 |
| 课程已被删除 | 显示404页 + 「返回课程列表」 |
| 操作无权限 | toast「没有操作权限」 |
