# 微课管理平台 · 设计系统 v1.1

> 依据: Element Plus 主题 / ui-ux-pro-max / WCAG 2.1 AA
> 角色差异化: 教师/管理员/教务处 = 教育专业感 · 学生 = 现代学习感 · 视频 = 沉浸式深色

## 8. 布局断点

| 断点 | 行为 |
|:-----|:-----|
| ≥ 1280px | 完整侧边栏 + 表格全列 |
| 1024-1279px | 折叠侧边栏 (64px) |
| 768-1023px | 折叠侧边栏 (64px) + 内容紧凑 |
| < 768px | 学生端单列 + 底部 Tab Bar（PC 端不显示 Tab Bar） |

> **PC/移动端 = 两套独立布局，不是响应式分支**

## 9. 组件规则

### 9.1 卡片
- 默认 `border-radius: var(--radius-md)` (8px)
- 学生端用 `var(--radius-lg)` (12px)
- 默认阴影 `var(--shadow-md)`，hover `var(--shadow-lg)` + `translateY(-2px)`

### 9.2 按钮
- 默认 `border-radius: var(--radius-sm)` (4px)
- 学生端用 `var(--radius-md)` (8px)
- hover/active 必须有过渡动画
- **必须**有 `cursor: pointer`

### 9.3 表格
- `border-radius: var(--radius-md)` + `overflow: hidden`
- 表头背景 `#f5f7fa`，字重 600

### 9.4 弹窗
- `border-radius: var(--radius-md)` (8px)
- 宽度 **≤ 600px**（表格嵌入除外）
- 移动端 `width: 90%`

### 9.5 表单
- label 字重 500
- input/select 圆角 4px
- 学生端圆角 8px

### 9.6 标签 (tag)
- 圆角 4px
- **状态色三件套**：背景浅 + 文字主色

### 9.7 进度条
- 圆角 4px
- `transition: width 0.3s ease`

## 10. 禁止项

- ❌ `style="..."` 内联样式 → 用 `class=""`
- ❌ `!important` → 用更高优先级选择器
- ❌ `color: #000` / `color: #fff` → 用 `#303133` / `#f5f5f5`
- ❌ placeholder 替代 label → 必须用 `<label>` 或 `aria-label`
- ❌ 弹窗宽度 > 600px → 表格嵌入除外
- ❌ 表格固定高度 → 用 `max-height`
- ❌ emoji 作为图标 → 用 Element Plus SVG 图标
- ❌ **PC 端显示底部 Tab Bar** → Tab Bar 仅移动端
- ❌ 学生端 PC 复用移动端样式
- ❌ 移动端用 PC 复杂布局

---

## 11. 无障碍 (a11y)

- 所有交互元素必须有 `cursor: pointer`
- 焦点状态可见（默认 Element Plus focus ring）
- 文本对比度 ≥ 4.5:1
- 尊重 `prefers-reduced-motion`
- 所有图标按钮必须有 `aria-label`
